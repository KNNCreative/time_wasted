# Time Wasted
Chrome extension to track how much time wasted on Social Networking Sites, currently support Facebook

![Demo](demo.gif "Demo")

## Download
https://chrome.google.com/webstore/detail/time-wasted/fahkhelepljjjcgekgbdmmbckpmclncl

## Contribute
    # install packages
    $ npm install

    # build extension
    $ gulp build
